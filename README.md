# Movie-Recommendation

**Overview**
This project implements a movie recommendation system to suggest movies to users based on their preferences and ratings. It uses collaborative filtering and content-based filtering approaches, along with matrix factorization techniques to provide personalized recommendations.

**Techniques Used**
Collaborative Filtering: Matrix factorization techniques such as Singular Value Decomposition (SVD).
Content-Based Filtering: Using movie metadata (genres, director, actors) to recommend similar movies.
Evaluation Metrics: RMSE (Root Mean Squared Error) and Precision-Recall.

**Dataset**
The dataset consists of:

Movie information: titles, genres, release year.
User ratings: user ID, movie ID, rating, timestamp.
